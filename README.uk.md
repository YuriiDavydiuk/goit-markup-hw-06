# 🎨 WebStudio Landing

Навчальний проект лендинг-сторінки веб-студії, розроблений під час проходження курсу з HTML та CSS.

## 📋 Про проект

**WebStudio Landing** — це сучасна і привабливо оформлена лендинг-сторінка, яка презентує послуги веб-студії. Проект демонструє практичне застосування знань з HTML та CSS для створення адаптивного та привабливого веб-дизайну.

### Основні особливості:
- ✨ Чистий та читаючий HTML код
- 🎯 Стильне оформлення з використанням CSS
- 📱 Адаптивний дизайн
- 🔗 Навігаційне меню
- 🎬 Привабливий hero-секція з call-to-action кнопкою
- 📞 Контактна інформація

## 🚀 Швидкий старт

### Вимоги
- Будь-який сучасний браузер (Chrome, Firefox, Safari, Edge)
- Текстовий редактор для редагування коду (VS Code, Sublime Text, Atom)

### Встановлення

1. Клонуйте репозиторій:
```bash
git clone https://github.com/your-username/webstudio-landing.git
```

2. Перейдіть до папки проекту:
```bash
cd webstudio-landing
```

3. Відкрийте файл `index.html` у браузері подвійним кліком або через контекстне меню.

## 📁 Структура проекту

```
webstudio-landing/
├── index.html          # Основний HTML файл
├── styles/
│   └── styles.css      # Стилі проекту
├── images/             # Папка з зображеннями
└── README.md           # Цей файл
```

## 📝 Структура HTML

Проект містить такі основні секції:

- **Header** — навігаційне меню та логотип
- **Hero Section** — привітна секція з головним заголовком та кнопкою "Order Service"
- **Footer** — контактна інформація
