# 🎨 WebStudio Landing

An educational landing page project for a web studio, developed during an HTML and CSS course.

## 📋 About the Project

**WebStudio Landing** is a modern and beautifully designed landing page that showcases web studio services. This project demonstrates the practical application of HTML and CSS knowledge for creating a responsive and attractive web design.

### Key Features:
- ✨ Clean and readable HTML code
- 🎯 Stylish design using CSS
- 📱 Responsive design
- 🔗 Navigation menu
- 🎬 Attractive hero section with a call-to-action button
- 📞 Contact information

## 🚀 Quick Start

### Requirements
- Any modern browser (Chrome, Firefox, Safari, Edge)
- A text editor for code editing (VS Code, Sublime Text, Atom)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/webstudio-landing.git
```

2. Navigate to the project folder:
```bash
cd webstudio-landing
```

3. Open the `index.html` file in your browser by double-clicking it or through the context menu.
